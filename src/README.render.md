# LiveEvents API - Деплой на Render.com

## Подготовка к деплою

1. Убедитесь, что все файлы находятся в репозитории:
   - `Dockerfile` - для контейнеризации приложения
   - `render.yaml` - для автоматической настройки на Render
   - Исходный код приложения

## Процесс деплоя

1. Зарегистрируйтесь на [Render.com](https://render.com)
2. Создайте новый Web Service, подключив ваш репозиторий
3. Render автоматически обнаружит `Dockerfile` и начнет сборку
4. Настройте переменные окружения в интерфейсе Render:
   - `DatabaseOptions__ConnectionString` - строка подключения к базе данных
   - `JwtOptions__Key` - ключ для JWT аутентификации
   - `MailOptions__Password` - пароль для почтового сервиса (если используется)
5. Дождитесь завершения сборки и развертывания

## Конфигурация

- Приложение будет доступно по предоставленному Render URL
- Health check endpoint: `/health`
- Порт: автоматически назначается Render
